About Me

I’m Gajula Eshwar, an entry-level IT professional passionate about building dynamic web applications and creating innovative solutions through web development, machine learning, and IoT. With a solid foundation in Computer Science and Engineering, I thrive in environments that challenge my problem-solving skills and push me to learn continuously.

Over the past few years, I’ve honed my expertise as a Front End Developer and IoT Developer, crafting projects that range from intuitive web interfaces to real-time sensor monitoring systems.

At Code Clause, I was responsible for developing interactive web applications like an Event Management System, Countdown Timer, Background Generator, and Simple Calculator, utilizing cutting-edge technologies like HTML, CSS, JavaScript, React JS, and Bootstrap. This role helped me deepen my understanding of responsive design and user experience (Jul 2024 - Aug 2024).

Prior to that, I worked with Coding Raja Technologies where I built a Resume Generator web application, bringing together HTML, CSS, Bootstrap, and JavaScript to offer users a seamless experience in creating custom resumes (Mar 2024 - Apr 2024).

In addition to web development, I have hands-on experience in IoT Development, having worked with Bolt IoT. I successfully designed a smart monitoring system, integrating multiple sensors for automated alerts based on light and temperature readings. My work focused on real-time data collection and analysis, empowering users with actionable insights through IoT modules (Nov 2023 - Dec 2023).
